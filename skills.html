مهاراتي وتقنياتي التفصيلية

هذه الصفحة تعرض مهاراتي بشكل تفصيلي، سواء ما أتقنه بالفعل أو ما أخطط لتعلمه في المستقبل القريب. أسعى دائمًا لتطوير نفسي وإضافة مهارات جديدة لخدمة عملائي بشكل أفضل.

تطوير الواجهات الأمامية (Front-End Development)

HTML5

إنشاء هيكل صفحات المواقع بأسلوب منظم وسليم، باستخدام عناصر حديثة مثل:

section article header footer nav form
مستوى المهارة

CSS3

تنسيق التصميمات والتحكم الكامل في شكل الموقع، مع إتقان:

  • استخدام Flexbox وGrid في توزيع العناصر
  • استخدام الـ Transitions وAnimations (بما فيها animation-delay, animation-duration, keyframes) لعمل تأثيرات بصرية جذابة
  • التعامل مع responsive design (تصميم متجاوب) باستخدام media queries
مستوى المهارة

Bootstrap 5

مكتبة جاهزة تساعدني في بناء تصميمات احترافية بسرعة وسهولة:

  • التعامل مع الـ Grid System
  • استخدام المكونات الجاهزة مثل Buttons, Cards, Navbar, Modals
مستوى المهارة

JavaScript

أساسيات جيدة جدًا، ومهارات في:

  • التعامل مع الـ DOM (للربط بين التصميم والبرمجة)
  • استخدام الأحداث (events) مثل click وinput وغيرها
  • استخدام setTimeout, setInterval
  • بناء مشروعات بسيطة كاملة، مثل آلة حاسبة
  • استخدام بيانات وهمية (fake API) بصيغة JSON وربطها مع الموقع

مكتبات وتقنيات إضافية

Figma to Web

تحويل تصميمات Figma إلى كود حقيقي (HTML + CSS + JS)، يعني لو العميل عنده تصميم على فيجما، أقدر أبني موقع مطابق 100%.

ThimWagon

منصة بأستخدمها لتحميل قوالب مجانية وتعديلها بما يناسب احتياجات المشروع.

Animate.css

مكتبة جاهزة فيها تأثيرات مثل fade in/out، bounce، slide، zoom وهكذا.

Google Fonts

استخدام خطوط احترافية من جوجل عشان تحسن شكل الكتابة في الموقع.

مشروعات منجزة

حاسبة أبو عمارة الخشب

  • تحسب المتر المكعب والسعر تلقائيًا
  • نسخة تعمل أونلاين (Web App)
  • استخدمت فيها HTML, CSS, JavaScript، ورفعتها على GitHub
  • فيها input ديناميكي، وحاولت تعملها PWA (تشغيل أوفلاين)
عرض المشروع

خطة التعلم المستقبلية

تطوير الواجهات الخلفية (Back-End) - جاري التعلم

Node.js

  • بناء السيرفر
  • عمل API
  • التعامل مع قواعد البيانات (مثلاً MongoDB لاحقًا)
  • إرسال واستقبال البيانات من الموقع

Express.js

إطار عمل خفيف يساعدني في بناء API بطريقة منظمة

قواعد البيانات

  • MongoDB
  • MySQL (كخيار ثاني)

أطر عمل متقدمة

  • React
  • Git & GitHub بشكل أعمق
  • PHP (كمسار ثاني لو احتجت، ليس أولوية حاليًا)

البنية التحتية والشبكات

  • أساسيات شبكات الحاسوب (من المبتدئ للمتقدم)
  • شهادة CCNA (Cisco Certified Network Associate)
  • إدارة خوادم لينكس
  • نشر واستضافة السيرفرات (Apache, Nginx وغيرها)

الأمن السيبراني

  • أساسيات الأمن السيبراني والاختراق الأخلاقي
  • مفاهيم أمن الشبكات وجدران الحماية
  • أساسيات اختبار الاختراق
