نظام إدارة المستخدمين وتسجيل الدخول

نظام إدارة مستخدمين تفاعلي يتيح للمستخدمين تسجيل الدخول، عرض بياناتهم الشخصية، إدارة المنشورات، وعرض قوائم المستخدمين بطريقة احترافية ومتجاوبة.

وصف المشروع

نظام إدارة المستخدمين هو تطبيق ويب تم تطويره لمحاكاة عملية تسجيل الدخول وإدارة الحسابات مع لوحة تحكم متكاملة. يوفر النظام واجهة مستخدم نظيفة وسهلة الاستخدام، مع ميزات مثل عرض البيانات الشخصية (الاسم، البريد الإلكتروني، الهاتف، إلخ)، إدارة المنشورات الخاصة بالمستخدم، وعرض قوائم المستخدمين مع تفاصيلهم ومنشوراتهم. تم تصميم النظام ليكون متجاوبًا مع جميع الأجهزة، مع التركيز على الأداء العالي وتجربة المستخدم الممتازة. يعتمد هذا النظام بشكل كبير على واجهة برمجة تطبيقات (API) وهمية (يمكنك الاطلاع على ملفات JSON الخاصة بها في صفحة API) لمحاكاة التفاعل مع قاعدة بيانات، مما يوضح القدرة على بناء تطبيقات تعتمد على جلب البيانات من مصادر خارجية.

المميزات الرئيسية

  • واجهة تسجيل دخول آمنة مع تحقق وهمي باستخدام واجهة برمجية (API) خارجية.
  • لوحة تحكم متكاملة تعرض البيانات الشخصية، المنشورات، وقوائم المستخدمين.
  • تصميم متجاوب يعمل بسلاسة على الهواتف المحمولة، الأجهزة اللوحية، والحواسيب.
  • دعم الوضع الليلي لتحسين تجربة القراءة في الإضاءة المنخفضة.
  • تأثيرات تفاعلية مثل رسائل التحميل الديناميكية ومعالجة الأخطاء باستخدام JavaScript.
  • محاكاة جلب بيانات المستخدمين والمنشورات من API وهمي.

استخدام واجهة برمجية (API)

النظام يعتمد على واجهة برمجية وهمية (JSONPlaceholder API) لمحاكاة استرجاع البيانات ديناميكيًا. يمكنك رؤية هيكل البيانات المستخدمة في ملفات JSON المتاحة في صفحة API.

  • تسجيل الدخول: يتم استخدام `https://jsonplaceholder.typicode.com/users` للتحقق من بيانات المستخدم. يتم مقارنة اسم المستخدم المدخل مع حقل `username` وكلمة المرور مع حقل `id` في بيانات الـ API.
  • إدارة المنشورات: يتم استرجاع المنشورات باستخدام `https://jsonplaceholder.typicode.com/posts`، حيث يتم عرض المنشورات الخاصة بالمستخدم المسجل دخوله بناءً على `userId`.
  • قوائم المستخدمين: يتم استرجاع قائمة المستخدمين من نفس واجهة الـ API لعرض تفاصيلهم (مثل الاسم، البريد الإلكتروني، العنوان) ومنشوراتهم.

تم تصميم التفاعل مع الـ API باستخدام JavaScript (Fetch API) لضمان استرجاع البيانات بشكل غير متزامن، مع معالجة الأخطاء (مثل رسائل "حدث خطأ أثناء تحميل البيانات") لتحسين تجربة المستخدم.

كيفية استخدام النظام

لتجربة النظام، يمكنك استخدام بيانات تسجيل الدخول التالية (مستمدة من JSONPlaceholder API):

  • اسم المستخدم: Bret, كلمة المرور: 1
  • اسم المستخدم: Antonette, كلمة المرور: 2
  • اسم المستخدم: Samantha, كلمة المرور: 3
  1. تسجيل الدخول: افتح صفحة تسجيل الدخول، وأدخل اسم المستخدم وكلمة المرور (مثال: "Bret" و"1").
  2. لوحة التحكم: بعد تسجيل الدخول، يتم توجيهك إلى لوحة التحكم حيث يمكنك عرض بياناتك الشخصية (البريد الإلكتروني، الهاتف، العنوان، إلخ).
  3. إدارة المنشورات: انتقل إلى قسم "منشوراتي" لعرض المنشورات الخاصة بك، مع رسائل تحميل ديناميكية.
  4. إدارة المستخدمين: في قسم "المستخدمين"، يمكنك عرض قائمة المستخدمين، تفاصيلهم، ومنشوراتهم.
  5. الوضع الليلي: استخدم زر تبديل الوضع الليلي في صفحة تسجيل الدخول لتحسين تجربة الاستخدام.
