موقع كورساتي

موقع "كورساتي" هو منصة تعليمية تفاعلية شبيهة بـ Udemy، تم تصميمها وتطويرها لعرض الكورسات وإدارة الحسابات. يتميز الموقع بواجهة مستخدم جذابة وسهلة الاستخدام، مع ميزات ديناميكية تحاكي بيئة تعليمية متكاملة. يعتمد هذا المشروع على واجهة برمجية (API) وهمية (يمكنك الاطلاع على ملف JSON الخاص بها في صفحة API) لمحاكاة استرجاع بيانات الدورات، مما يوضح القدرة على بناء تطبيقات تعتمد على مصادر بيانات خارجية.

المميزات الرئيسية

  • واجهة مستخدم تفاعلية ومتجاوبة بالكامل مع جميع الأجهزة.
  • لوحة تحكم إدارية (Admin Dashboard) متكاملة لإدارة الكورسات والمستخدمين.
  • استخدام واجهة برمجية (API) وهمية لمحاكاة استرجاع البيانات ديناميكيًا.
  • نظام تسجيل دخول مع حساب إداري مخصص لتجربة جميع الميزات.
  • عرض ديناميكي للدورات التدريبية بناءً على بيانات JSON.

كيفية تجربة لوحة الإدارة

للوصول إلى لوحة التحكم الإدارية واستكشاف وظائفها، يمكنك استخدام بيانات تسجيل الدخول التالية:

  • البريد الإلكتروني: admin@example.com
  • كلمة المرور: Admin123

بعد تسجيل الدخول، يمكنك استكشاف لوحة التحكم وتجربة إضافة الكورسات. يرجى ملاحظة أن صفحات الكورسات الفردية لم يتم إنشاؤها، حيث أن البيانات وهمية وتهدف إلى محاكاة الوظائف الأساسية للنظام.

التقنيات المستخدمة

  • HTML5 و CSS3: لتصميم الواجهة الأمامية الجذابة.
  • JavaScript: لإضافة التفاعلية وإدارة منطق الواجهة، بما في ذلك التعامل مع الـ API الوهمي.
  • Bootstrap: لضمان التوافق مع جميع الأجهزة وتوفير تصميم متجاوب.
  • API وهمي (JSON): لمحاكاة استرجاع البيانات وعرضها ديناميكيًا.
