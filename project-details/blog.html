المدونة الشخصية (نموذج ديناميكي)

المدونة الشخصية هي منصة ديناميكية تم تطويرها لعرض المقالات والمحتوى بطريقة منظمة وجذابة. يتيح هذا المشروع للمستخدمين تصفح المقالات والبحث فيها بسهولة، مما يوفر تجربة قراءة ممتعة وفعالة. تعتمد المدونة على ملف JSON وهمي (يمكنك الاطلاع عليه في صفحة API) لمحاكاة استرجاع المقالات ديناميكيًا، مما يوضح القدرة على التعامل مع البيانات الخارجية وعرضها بشكل تفاعلي.

المميزات الرئيسية

  • تصميم ديناميكي يسمح بإضافة مقالات جديدة وتحديث المحتوى بسهولة.
  • واجهة مستخدم نظيفة وبديهية ومتجاوبة مع جميع الأجهزة.
  • خاصية البحث عن المقالات باستخدام JavaScript لتحسين إمكانية الوصول للمحتوى.
  • تنسيق جذاب لعرض المقالات مع صور مميزة لجذب القارئ.
  • محاكاة استرجاع البيانات من API وهمي لعرض المقالات.

التقنيات المستخدمة

  • HTML5 و CSS3: لتصميم هيكل وواجهة المستخدم العصرية.
  • JavaScript: لإدارة المحتوى الديناميكي وتوفير التفاعلية، بما في ذلك وظيفة البحث وتحميل البيانات.
  • Bootstrap: لضمان التوافق الكامل مع جميع أحجام الشاشات وتوفير تصميم متجاوب.
  • JSON (ملف وهمي): لمحاكاة مصدر البيانات للمقالات.
