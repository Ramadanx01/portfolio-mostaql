ملفات JSON المستخدمة في المشاريع

هذه الصفحة تعرض ملفات JSON التي تم استخدامها لمحاكاة البيانات في بعض المشاريع الديناميكية. هذه الملفات تساعد في فهم كيفية هيكلة البيانات التي يتم استرجاعها من واجهات برمجة التطبيقات (APIs).

بيانات المستخدمين (Login System)

هذا الملف يمثل هيكل بيانات المستخدمين المستخدم في مشروع نظام تسجيل الدخول الوهمي. يحتوي على معلومات أساسية لكل مستخدم مثل الاسم، اسم المستخدم، البريد الإلكتروني، والعنوان.

بيانات المنشورات (Login System)

هذا الملف يمثل هيكل بيانات المنشورات المستخدم في مشروع نظام تسجيل الدخول الوهمي. كل منشور مرتبط بمعرف مستخدم (`userId`) ويحتوي على عنوان ومحتوى.

بيانات الدورات (Korsaty)

هذا الملف يمثل هيكل بيانات الدورات التدريبية المستخدم في مشروع "كورساتي". يحتوي على تفاصيل الدورات مثل العنوان، الوصف، الصورة، والتقنيات المستخدمة.
